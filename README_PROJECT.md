# 🤖 AI Resume Analyzer

An AI-powered resume analyzer built with **Claude API** that extracts skills, scores job matches, and gives career recommendations in seconds.

## ✨ Features
- 📄 Paste any resume → get instant AI analysis
- 🎯 Job match scoring against a job description
- ⭐ Top skills extraction
- 💪 Strengths & improvement areas
- 🚀 Recommended job roles
- 🌐 Clean web UI included

## 🛠️ Tech Stack
`Python` `Claude API` `Anthropic SDK` `Flask`

## 🚀 Quick Start

```bash
git clone https://github.com/pradeepb-pixel/ai-resume-analyzer
cd ai-resume-analyzer
pip install -r requirements.txt
cp .env.example .env
# Add your Anthropic API key to .env
python main.py        # CLI version
python app.py         # Web UI version → open localhost:5000
```

## 📸 How It Works
1. Paste your resume text
2. Optionally paste a job description
3. Claude AI analyzes and returns structured insights
4. Get your job match score, skills, and recommendations
